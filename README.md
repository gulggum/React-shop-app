## 프로젝트 소개

- React + Vite로 개발 중
- 네비게이션 바 구현
- 다크/화이트 테마 토글 기능 예정

## 사용 기술

- React
- Vite
- Styled-components
- Recoil (상태 관리 예정)

## 프로젝트 폴더/파일 구조

```plaintext
react-shop/
├─ public/
├─ src/
│ ├─ assets/ # 이미지, 아이콘 등
│ ├─ components/ # 재사용 가능한 컴포넌트
│ │ ├─ Header/
│ │ └─ Button/
│ ├─ pages/ # 페이지 단위 컴포넌트
│ │ └─ Home.jsx
│ ├─ recoil/ # 상태 관리 파일
│ │ └─ themeAtom.js
│ ├─ App.jsx
│ └─ main.jsx
├─ package.json
└─ vite.config.js
```
